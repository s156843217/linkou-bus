# 林口公車路線地圖

輸入社區或地址，在地圖上看到附近的公車站牌、有哪幾條路線經過，點路線可看完整走向與沿途站牌；也可直接輸入公車路線編號查整條走向。

太平洋房屋・林口捷運加盟店 客戶服務工具。

## 技術
- 純前端（HTML + CSS + 原生 JS），無建置流程，直接開 `index.html` 即可。
- 地圖：Leaflet + OpenStreetMap 圖磚。
- 站牌定位：沿用林口社區／門牌座標資料（`linkou-data.js`）。

## 資料來源
- 公車站牌與路線：交通部 **TDX 運輸資料流通服務平台**（市區公車／公路客運），會定期更新。
- 資料由 `tools/fetch-bus.ps1`（在主專案）抓取、以 Douglas–Peucker 抽稀線型後產生 `bus-data.js`。

## 更新資料
於主專案執行 `tools/fetch-bus.ps1` 重新產生 `bus-data.js`，再覆蓋本 repo 的同名檔並 push 即可。

> 本工具僅供參考，實際發車、停靠與路線異動請以各客運業者及站牌公告為準。
